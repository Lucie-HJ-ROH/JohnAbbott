3 ways to create CSS
=====================
1. inline using style attribute

Inline styles are CSS styles that are applied to one element using the style attribute.
For example, 
in html
<element_name style = "css rule">element_content</element_name>
<p style = "color:blue">Welcome</p>
<p style = "color:blue; background-color: white;">Hello</p>


2. internal using <style> element
An internal (embedded) stylesheet contains the CSS rules that apply to the a single webpage. 
Yoy create an internal CSS by using the <style> element in the head section of the particular file. 

For example,
in html
in head section
<head>
   <style type="text/css">
	p {
	    color:blue;
	    background-color:yellow;
	  }
   </style>


3. external (create a css file and link it to the html file)

The External Stylesheet is a seperate document with an extension .css. 
You use <link type="text/css" rel="stylesheet" href="styles/filename.ccs" /> element to link your website. 

For example,
	3a. create a new css document and save it as filename.css
	3b. in html document, link to the css document
Note: External CSS does not contain any html element

in head section
<head>
   <link type="text/css" rel="stylesheet" href="styles/filename.css" />

Syntax of a CSS rule 
====================
selector { style declaration }
selector {property_name: value;}

combining selector (when style declarations are the same)
h1 {color:blue;}
p {color: blue;}
	h1, p {color:blue;}
combining style declarations (when selectors are the same)
p { color: red;}
p {background-color:white;}  
	p { color:red; 
	    background-color:white;}

1) Use classes to style several elements
2) Different between dependent and independent classes
	.class_name {style declaration}
	selector.class_name { style declaration }
	p.intro {font-size:20px;}
3) Use ID selector 
	#Id_name {style declarations}
	selector#Id_name {style declarations}
	<p id="warning" class="intro intro2">

1. Fonts Category: serif, sans-serif, monospace, cursive, and fantasy
a)	font-family: Arial, Helvetica, sans-serif

	body {
		font-family: Arial, Helvetica, sans-serif;
	}

	h1,h2,h3 {
		font-family: Helvetica;
	}
b) font-size:length
 where length: i) absolute ii) relative, or iii) percentage
	i) Absolute unit of measurments: mm (millimeter), cm, in (inch), pt (point), pc (pica 1/6 in)
	ii)Relative: px (pixel), em (width of a capital letter M, or any capital letter), ex (height of the lower case x), rem (root)
	iii) percentage: font-size:150%

Controlling spacing and indentation
Kerning - space between letters
	letter-spacing:value
Tracking - space between words 
	word-spacing:value
Leading - space between lines
	line-height: value

Setting the style, weight, and decorative features
font-style: type
where type: normal, italic, oblique

font-weight: bold, bolder, normal, lighter, and a value between 400 to 900 

text-decoration: type
where type: none, underline, overline, line-through, blink

<p><a href="msn.com><strong>msn</strong></a></p>

in css
a:link {
      font-weight: bold;
	text-decoration: underline;
}

text-transform: type
where type: none, lowercase, uppercase, capitalize

font-variant - displays text in uppercase and small font
font-variant: normal, small-caps
